# LAAM Clone Store

Production ecommerce platform inspired by LAAM.pk

Features:

- Next.js
- PostgreSQL
- Prisma
- Admin Dashboard
- Products
- Categories
- Inventory
- Orders
- Stripe
- Easypaisa
- JazzCash
- Email Notifications
- Analytics
- Docker Deployment

## Local Development

```bash
npm install
```

```bash
cp .env.example .env
```

```bash
npm run db:push
```

```bash
npm run seed
```

```bash
npm run dev
```
